Computing (or computation) is done by computers. Perhaps we can say that a program is a plan for computation, and programming is making the plan. (We will avoid for the moment the question of what computation is.) The languages in which we express our plans is a 'programming language'. I have created a programming language, which I have named 'Goat'.
