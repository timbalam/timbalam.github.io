Making a programming language

This article discusses my motivation for making Goat, a new programming language. To find out about the language itself, some good places to start are the beginner's introduction at Learn, or the Examples.

DISCLAIMER: Goat is very incomplete! Creating a useful programming language is a big task. Currently, the language is significantly less useful than the most commonly used ones (and also many less commonly used ones). If you want to get stuff done, you should look at those languages instead.

Computer programs are powerful tools for automation. However, going about making a program is a complicated endeavour! I'm interested in exploring where this complexity comes from. Is programming complicated due to the languages we use? Can we make it easier? Computers — What do they do? Do they do things? Let's find out! Since programming languages are the "user interface" for programming, it seems a reasonable place to start investigating. I made a programming language Goat to experiment with some ideas I have for making programming and programs easier to learn and to write. In summary, Goat seeks to make it easier to:

  • Learn programming: a small core of concepts with well-defined interactions that lets you quickly grasp the basics move onto higher level programming concepts.
  • Write programming: simple and flexible syntax that lets you write your ideas with minimal syntactic hurdles; interactive program-writing tools that let you incrementally develop and test ideas.
  • Learn programs: readable syntax with precise meaning that lets you work out what a program does.
  • Write programs: modular programs with precise interfaces that let you abstract complex behaviour and build upon reliable foundations.

Goat is still a work in progress and it remains to be seen whether it will live up to these lofty ideals.

Learning Goat shouldn't require previous experience with other programming languages — Goat is described thoroughly in its own terms in the beginners introduction and the examples. If I've got you curious to know more, these are the places to go next!
