Getting started with my language

The easy way (on Windows at least):

  • Download the binary for the latest GitHub release
  • Unpack the archive and place myi-exe.exe somewhere.
  • Use the command prompt to cd to the directory that you placed myi-exe.exe.
  • Alternatively, place myi-exe.exe in a directory mentioned in your %PATH% (type echo %PATH%)
  • To interpret a text file code.my type myi-exe.exe code.my.

Alternative way (if easy way doesn't work, or you want to play with the source code):

  • You will need to install Stack on your system
  • If you have git installed you can clone the git repo from the shell via git clone https://github.com/timbalam/myi
  • Alternatively use a browser to visit https://github.com/timbalam/myi and select the green "Clone or download" button to the far right of the screen
  • Press "Download ZIP" and save the archive
  • Unpack the archive and place the source folder somewhere
  • Either way, navigate to the source folder in your shell or command prompt
  • Type "stack build" to build the source
  • To interpret a text file code.my type stack exec myi code.my.
