Getting started with my language

The easy way (on Windows at least):

  • Download the binary for the latest GitHub release to your usual download folder.
  • Open the file explorer program on your computer and navigate to the folder with the downloaded archive. Right-click and unpack the archive and place the contained file myi-exe.exe somewhere, for example in your home directory, 'Documents' directory, or a create a new folder.
  • Open the command prompt or terminal interface on your computer, type cd followed by the path to the directory where you placed myi-exe.exe, and hit 'enter'.
  • Alternatively, place myi-exe.exe in a directory mentioned in your %PATH% (to see a list of directories, type echo %PATH% and hit 'enter').
  • To test that the my language interpreter is installed correctly, you can type myi-exe.exe --version and hit 'enter'. If you see something like: myi-exe.EXE: --version: openFile: does not exist (No such file or directory) then congratulations! It works! If you see something like: 'mye-exe.exe' is not recognized as an internal or external command, operable program or batch file.', unfortunately something hasn't worked. Perhaps try the alternative way below.

Alternative way

If easy way doesn't work, or you want to play with it, you can build the source code:

  • You will need to install Stack on your system.
  • If you have git installed, you can use the command prompt or terminal command to clone the myi git repository (containing the program source code). Type git clone https://github.com/timbalam/myi and hit 'enter' to copy the repository folder to the current directory.
  • Alternatively you can obtain the git repository using a web browser with the following steps:
    • Open a web browser, visit https://github.com/timbalam/myi and select the green "Clone or download" button to the far right of the screen.
    • Press "Download ZIP" and save the archive to your usual downloads folder.
    • Find the downloaded archive in your file explorer program. Right-click and unpack the archive and place the source folder somewhere (e.g. your home directory, 'Documents' directory, or create a new directory).
  • Either way, navigate to the source folder in the command prompt or terminal interface. Type cd followed by the path to the source folder, and hit 'enter'.
  • Type stack build and hit 'enter' to build the source. If Stack is successful you should eventually see a message like: 
    myi-0.1.0.0: copy/register
...
Installing executable(s) in
...
Registering myi-1.0.0...

Writing and running your code

You can write your code in any text file editor, as long as it can save files in plain text format. For example Notepad++ is a good option if you use Windows, and another is Atom which is available on most operating systems. Open the text editor, type your code, and save the file in the same directory that you installed myi-exe.exe.

To interpret the code, return to your terminal or command line prompt and type myi-exe.exe followed by the name you used to save your code file and hit 'enter'. If you installed via Stack (the 'Alternative way' above), you will need to type stack exec myi-exe -- (don't forget the two '-' dash characaters) follwed by the name of your saved code file, and hit 'enter'.
